# dotfiles
ML4W customized dotfiles for Hyprland

<h1>installation:</h1>

1. <code>yay -S ml4w-hyprland</code>

2. <code>ml4w-hyprland-setup</code>

3. <code>cp -r dotfiles/* ~/.config/</code>
